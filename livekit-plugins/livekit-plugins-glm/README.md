# LiveKit Plugins Zhipu GLM

Agent Framework plugin for services from Zhipu. 

## Installation

```bash
pip install livekit-plugins-glm
```

## Pre-requisites

You'll need an API key from Zhipu. It can be set as an environment variable: `OPENAI_API_KEY`
